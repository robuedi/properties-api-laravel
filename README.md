## Properties SPA Project: API side - work in progress...

Technologies used in the project:

- Laravel 10 
- PHP8.2

## How to use it

1. Install Docker
2. Run `sail up -d`
3. Run `sail composer install`
4. Run `sail artisan migrate:fresh --seed`

### API documentation 

Go to http://localhost/api/documentation#/


### PHPUnit

Run `sail test`
